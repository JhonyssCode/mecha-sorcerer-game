# Mecha Sorcerer - Mega Drive Project

Este projeto foi criado com a ajuda do Antigravity.

## Como Compilar
1. Certifique-se de que o **SGDK** está instalado.
2. Defina a variável de ambiente `GDK` para o caminho do SGDK.
3. No VS Code, use a extensão **Genesis-Code** e pressione `F5`.
4. Ou use o `makefile` do SGDK via terminal: `%GDK%\bin\make -f %GDK%\makefile.gen`.

## Estrutura
- `src/`: Código fonte C.
- `res/`: Recursos (Imagens, Sons, Sprites).
- `inc/`: Arquivos de cabeçalho (.h).
